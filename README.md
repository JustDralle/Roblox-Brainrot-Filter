# Roblox Brainrot Filter

A lightweight browser extension that filters Roblox game cards based on
user-defined keywords and manual blocks, helping reduce low-quality or unwanted
recommendations.

## Features

- Filter games by keyword (title and optional creator)
- Manual hide button on each game card (block by Game ID)
- Real-time enable/disable
- Hide or dim filtered games
- Works on Home, Discover, Charts and Search pages

## Privacy

This extension does not collect, transmit, or sell any personal data.
All settings are stored locally in the browser using Chrome sync storage.

No external servers are used.

## Installation (development)

1. Download or clone the repository
2. Open `chrome://extensions` or `edge://extensions`
3. Enable Developer Mode
4. Click "Load unpacked" and select the extension folder
